## 6) 핵심 관찰 포인트

### 아드닷컴의 핵심 포지셔닝
1. 광고 크리에이티브 데이터 수집
2. 경쟁사/상품/LP 기반 분석
3. 히트 패턴 추출
4. 생성AI로 배너/이미지/영상 제작
5. 팀 협업/공유까지 연결

### 특히 중요한 차별점
- `LP URL 입력 → 자동 해석 → 일본어 카피 포함 배너 생성`
- 단순 생성AI가 아니라, 광고 데이터베이스 기반의 생성 워크플로우
- 분석과 제작이 한 제품 안에서 이어짐

### 시장 시사점
- 3rd-party cookie 약화 / 개인정보 규제 강화 환경에서
- 타게팅 우위보다 크리에이티브 우위가 중요해지는 흐름
- 따라서 `분석 only`보다 `분석 + 생성 + variation 대량화`가 강한 구조

## 7) 우리 관점에서 봐야 할 기능 갭
